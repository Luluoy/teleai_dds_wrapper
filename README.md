# TeleAI DDS Wrapper
### 项目简介
`teleai_dds_wrapper` 是一个基于 **CycloneDDS (0.11.0)** 和 **Eclipse Iceoryx** 构建的高性能 Python DDS 封装库。

本项目专为高带宽数据传输设计（如原始图像大包），基于Iceoryx 共享内存（SHM）技术，支持在整个二层广播域下的透明数据传输。

### 环境要求
- **操作系统**: Ubuntu 22.04
- **Python 版本**: 推荐 3.8~3.10
- **系统依赖**: 脚本 `compile_thirdparty.sh` 会自动安装必要的系统库

### 安装流程

> **⚠️ 重要警告**: 
> 本库的路径在安装时会被硬编码进环境配置中。**请勿在安装后移动文件夹位置**。如果必须移动，请务必删除旧环境并重新执行安装流程。
